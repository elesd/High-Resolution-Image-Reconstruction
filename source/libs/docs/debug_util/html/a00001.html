Detailed Description

This file is the main file of debug. Use this module only with macros, like DEBUG_UTIL_TRACE, DEBUG_UTIL_ERROR, DEBUG_UTIL_DEBUG. For activate traces you have to define the DEBUG_UTIL_ACTIVATE, without it, this module has no effect.
