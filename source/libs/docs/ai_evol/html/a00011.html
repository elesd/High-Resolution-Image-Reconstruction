insert a new element in the queue. Insert the given element in the last place in the heap (e.g.: in the last level most right place). With swapping move the element in the correct place. The element will be copied with the copy function which was given during initialization.

void priority_queue_pop ( priority_queue_QUEUE *  queue)

Remove the first element in the queue. This element is the maximum element. After the deletition the last element (e.g.: in the last level the most right one) will be placed to the root place. After this with swapping the elements will be reordered.
